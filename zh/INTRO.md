# Mac 开发配置学习手册
>工欲善其事，必先利其器。

手册内容为「**如何让一部全新的 MacBook 快速完成开发环境配置**」，主要面向 Web 开发者。其中的指导，在 Mavericks 和 Yosemite 上有效。

* 本手册可以减少配置开发环境的烦恼。让自己认识一个全新的世界。
